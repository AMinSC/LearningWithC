#### built-in functions
- getchar() / putchar()
    - Buffered I/O
    - 값이 저장된 메모리 값을 읽거나 출력
- _getch() / _getche()
    - Non-buffered I/O
    - 키보드 입력 자체에 대한 **감지**
- gets() / gets_s() / fgets() / puts()

#### keyword
- buffered I/O
- device file (special file)
    - file name: CON(console device)
- 시큐어 코딩
    - 보안 문제가 발생하지 않도록 코드 수준에서 대응 하는 것이 중
